# Examples

This directory contains example notebooks and demonstrations for the Smart Farmer Assistant.

## Available Examples

### `demo.ipynb`
**Interactive Jupyter notebook showcasing all features of Smart Farmer Assistant**

Includes:
- 🌱 Crop recommendations (North & South regions)
- 🌾 Fertilizer advice for 9 major crops
- 🐛 Pest control tips
- 📅 Planting season guides
- ☁️ Weather-based farming advice
- 🐄 Livestock management tips
- 🏆 Achievement system demo
- 🎮 Simulated farmer journey

## Run Locally

### Prerequisites
- Python 3.6+
- Jupyter Notebook

### Installation

```bash
# Install Jupyter
pip install jupyter

# Navigate to repository
cd Smart-Farmer-Assistant-

# Start Jupyter
jupyter notebook examples/demo.ipynb
```

### Running

```bash
# Open the notebook and run cells with Shift+Enter
# Or run all cells: Cell → Run All
```
